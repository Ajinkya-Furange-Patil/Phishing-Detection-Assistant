========================================
TROUBLESHOOTING
========================================

Problem: "No module named 'datasets'"
Solution: pip install datasets

Problem: "FileNotFoundError: datasets/enron"
Solution: Download Module 1 datasets first

Problem: "All scores are 0"
Solution: Check email text extraction - print sample text

Problem: "Out of memory"
Solution: Reduce sample size in process script
