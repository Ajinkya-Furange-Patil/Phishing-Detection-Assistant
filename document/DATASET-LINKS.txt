========================================
PHISHING DETECTION - DATASET QUICK LINKS
========================================

MODULE 1: EMAIL CONTENT ANALYSIS
---------------------------------
✓ Enron Corpus (ESSENTIAL)
  → https://huggingface.co/datasets/enron_emails
  → https://www.kaggle.com/datasets/wcukierski/enron-email-dataset
  → Python: pip install datasets; load_dataset("enron_emails")

✓ Nazario Phishing (ESSENTIAL)
  → http://monkey.org/~jose/phishing/
  → Files: phishing0.mbox, phishing1.mbox, phishing2.mbox, phishing3.mbox
  → Alternative: https://github.com/najarianb/phishing-corpus

✓ LLM-Generated Phishing (ESSENTIAL)
  → https://github.com/FrancescoGreco/AI-Generated-Phishing-Detection
  → git clone https://github.com/FrancescoGreco/AI-Generated-Phishing-Detection.git

✓ TREC Spam Corpus (for MeAJOR alternative)
  → https://plg.uwaterloo.ca/~gvcormac/treccorpus/
  → TREC 2005, 2006, 2007 available


MODULE 2: URL ANALYSIS
----------------------
✓ ISCX-URL2016 (ESSENTIAL)
  → https://www.unb.ca/cic/datasets/url-2016.html
  → Kaggle: https://www.kaggle.com/datasets/sid321axn/malicious-urls-dataset

✓ PhiUSIIL (ESSENTIAL)
  → https://archive.ics.uci.edu/dataset/967/phiusiil+phishing+url+dataset
  → Direct: https://archive.ics.uci.edu/static/public/967/phiusiil+phishing+url+dataset.zip
  → Python: pip install ucimlrepo; fetch_ucirepo(id=967)

✓ PhishTank (Current URLs)
  → https://www.phishtank.com/
  → API: https://www.phishtank.com/api_info.php
  → CSV: https://data.phishtank.com/data/online-valid.csv

✓ OpenPhish
  → https://openphish.com/
  → Feed: https://openphish.com/feed.txt


MODULE 3: HEADER ANALYSIS
--------------------------
✓ Nazario .mbox (REUSE from Module 1)
  → Same as above - preserves headers

✓ Spam Genuine Mail Contents (ESSENTIAL)
  → https://www.kaggle.com/datasets/isuranga/spam-genuine-mail-contents-dataset
  → Kaggle CLI: kaggle datasets download -d isuranga/spam-genuine-mail-contents-dataset


MODULE 4: ATTACHMENT ANALYSIS
------------------------------
✓ Simulated Email Dataset (ESSENTIAL)
  → https://www.kaggle.com/datasets/vincentamonde/simulated-email-dataset
  → Search Kaggle: "simulated email attachment metadata"
  → Alternative: Create your own simulated metadata


MODULE 5: BEHAVIORAL PROFILING
-------------------------------
✓ SpaPhish (Framework only - Optional)
  → Search Google Scholar: "SpaPhish Spanish phishing dataset"
  → Use methodology, not necessarily the data

✓ Build from Module 1 datasets
  → Reuse Enron + Nazario
  → Extract behavioral features using custom lexicons


ESSENTIAL TOOLS
---------------
✓ Python Libraries:
  pip install datasets
  pip install ucimlrepo
  pip install kaggle
  pip install wget
  pip install pandas numpy scikit-learn torch transformers xgboost

✓ Kaggle Setup:
  1. Get API token: https://www.kaggle.com/<username>/account
  2. Download kaggle.json
  3. Place in ~/.kaggle/kaggle.json (Linux/Mac) or C:\Users\<user>\.kaggle\kaggle.json (Windows)
  4. chmod 600 ~/.kaggle/kaggle.json

QUICK START DOWNLOAD SEQUENCE
------------------------------
1. pip install datasets ucimlrepo kaggle
2. Download Enron: load_dataset("enron_emails")
3. wget http://monkey.org/~jose/phishing/phishing0.mbox
4. git clone https://github.com/FrancescoGreco/AI-Generated-Phishing-Detection.git
5. Download PhiUSIIL: fetch_ucirepo(id=967)
6. Setup Kaggle credentials
7. kaggle datasets download -d isuranga/spam-genuine-mail-contents-dataset
8. kaggle datasets download -d sid321axn/malicious-urls-dataset
